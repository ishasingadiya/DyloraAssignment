Employee Management System (React + Node + MongoDB)

Features:
- Add Employee
- Admin Login (JWT Authentication)
- View Employees
- Update Employee
- Delete Employee
- Protected Admin Routes

Tech Stack:
- Frontend: React, Bootstrap, Axios
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

How to Run:
1. Backend:
   cd back
   npm install
   npm start

2. Frontend:
   cd frontend
   npm install
   npm start
